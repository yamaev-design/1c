📋 Требования перед установкой

💻 Системные требования

  • 1С:Предприятие: платформа 8.3.20 или выше
  • Конфигурации: УТ 11.5+, УНФ 1.6+, КА 2.4+, ERP 2.5+
  • ОС: Windows 10/11, Server 2016+ или Linux (через wine)
  • Интернет: стабильное соединение для работы с ИИ
  • Права: права на изменение конфигурации 1С

🔐 Подготовка доступа

  • API-ключи маркетплейсов (WB, Ozon, Я.Маркет)
  • Доступы к ИИ-провайдерам (YandexGPT / GigaChat)
  • Права на выгрузку данных из 1С (товары, остатки, цены)
  • Бэкап базы 1С перед установкой модуля

⬇️ Установка модуля

1

Скачайте модуль

Перейдите в личный кабинет и скачайте актуальную версию модуля:

📦 Файл модуля
ai-seller-1c-v2.4.0.cf
📥

Кнопка «Скачать модуль» в разделе «Загрузки»

Рис. 1: Интерфейс личного кабинета для скачивания

2

Загрузите в 1С

В режиме «Конфигуратор» 1С выполните следующие действия:

  1. Меню Конфигурация → Загрузить внешнюю компоненту
  2. Выберите скачанный файл .cf
  3. Подтвердите загрузку и обновление конфигурации
  4. Дождитесь завершения процесса обновления
⚙️

Меню «Конфигурация» → «Загрузить внешнюю компоненту»

Рис. 2: Путь в меню 1С для загрузки модуля

✅ После загрузки перезапустите 1С в режиме «Предприятие»
3

Активируйте лицензию

В меню модуля в 1С:

  1. Откройте AI Seller → Настройки
  2. Введите лицензионный ключ из личного кабинета
  3. Нажмите «Активировать»
🔑 Пример ключа
AIS-XXXX-XXXX-XXXX-XXXX
ℹ️ Ключ привязывается к одному компьютеру. Для смены — обратитесь в поддержку.

⚙️ Настройка интеграций

🔗 Подключение маркетплейсов

Для работы с маркетплейсами необходимо добавить API-ключи:

  1. Откройте AI Seller → Интеграции
  2. Выберите нужную площадку (Wildberries / Ozon / Яндекс.Маркет)
  3. Вставьте API-ключ и нажмите «Сохранить»
  4. Проверьте статус подключения (должен быть зелёный)
⚠️ API-ключи хранятся в зашифрованном виде. Не передавайте их третьим лицам.

🤖 Настройка ИИ-провайдера

Выберите предпочтительного провайдера ИИ:

  • YandexGPT: лучше для русского языка, серверы в РФ
  • GigaChat: хорошая альтернатива, также серверы в РФ
  • Оба: модуль будет автоматически переключаться при ошибках

Для настройки:

  1. Откройте AI Seller → Настройки → ИИ-провайдеры
  2. Введите API-ключи выбранных провайдеров
  3. Нажмите «Проверить подключение»

✅ Проверка установки

Убедитесь, что модуль работает корректно:

1️⃣
Откройте меню модуля

В 1С должно появиться новое меню «AI Seller» в верхней панели.

2️⃣
Проверьте подключение

В Настройки → Статус должен быть зелёный индикатор «Подключено».

3️⃣
Запустите тестовую генерацию

Создайте тестовый товар и нажмите «Сгенерировать описание». Результат должен появиться за 10–30 секунд.

🎉

Установка прошла успешно!

Модуль готов к работе. Переходите к настройке интеграций.
