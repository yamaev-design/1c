🚀 Быстрый старт

Начните работать с DeepSite за 15 минут. Пошаговое руководство для новых пользователей.

Содержание

  • Шаг 1: Регистрация аккаунта
  • Шаг 2: Установка модуля в 1С
  • Шаг 3: Подключение маркетплейсов
  • Шаг 4: Первая генерация
  • Советы для новичков

Шаг 1: Регистрация аккаунта

  1. Перейдите на страницу активации триала
  2. Введите ваш email и номер телефона
  3. Подтвердите регистрацию через SMS или email
  4. Получите лицензионный ключ на почту
💡 Совет: Лицензионный ключ также доступен в личном кабинете после регистрации.

Шаг 2: Установка модуля в 1С

  1. Скачайте последнюю версию модуля из личного кабинета
  2. Запустите 1С в режиме предприятия
  3. Меню: Администрирование → Расширения и дополнения
  4. Нажмите «Установить из файла» и выберите скачанный файл .cfu
  5. Подтвердите установку и перезапустите 1С

Подробная инструкция по установке доступна в разделе Установка в 1С.

Шаг 3: Подключение маркетплейсов

  1. Откройте форму DeepSite в 1С (новая панель инструментов)
  2. Перейдите в настройки → Интеграции
  3. Выберите ваши маркетплейсы (WB, Ozon, Яндекс.Маркет)
  4. Введите API-ключи (инструкции для каждого МП в разделе Настройка интеграций)
  5. Нажмите «Проверить подключение»

Шаг 4: Первая генерация

  1. В списке товаров выделите 1-5 товаров для теста
  2. Нажмите кнопку «ИИ: Сгенерировать описание»
  3. Выберите шаблон промпта или создайте свой
  4. Дождитесь генерации (15-30 секунд на товар)
  5. Проверьте результат в предпросмотре
  6. Отредактируйте при необходимости и опубликуйте
✅ Готово! Вы успешно настроили DeepSite. Теперь можно обрабатывать товары пакетно.

Советы для новичков

  • Начните с малого: протестируйте на 10-20 товарах перед массовым запуском
  • Используйте готовые шаблоны промптов из библиотеки
  • Настройте авто-ответы на отзывы в первую очередь — это даст быстрый эффект
  • Включите уведомления в Telegram для мониторинга статуса генераций
  • Изучите часто задаваемые вопросы перед обращением в поддержку
