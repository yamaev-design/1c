Пример: получение списка товаров из Wildberries

# Запрос к нашему API (прокси к WB)
curl -X GET https://api.aiseller.ru/v1/marketplaces/wildberries/products \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "X-1C-Config: UT11" \
  -G \
  -d "limit=50" \
  -d "offset=0"

# Ответ:
{
  "products": [
    {
      "id": "12345678",
      "name": "Футболка мужская",
      "wb_id": "987654321",
      "price": 1490,
      "stock": 42,
      "rating": 4.8,
      "reviews_count": 156
    }
  ],
  "pagination": {
    "total": 234,
    "limit": 50,
    "offset": 0
  }
}

🔐 Безопасность и соответствие требованиям

🔒

Шифрование данных

TLS 1.3 для всех соединений. API-ключи хранятся в зашифрованном виде.

🏢

152-ФЗ и локализация

Персональные данные обрабатываются на серверах в РФ. Соответствие требованиям регуляторов.

🗂️

Разделение данных

Чувствительные данные (себестоимость, маржа) не покидают контур вашей 1С. В облако передаются только публичные данные.

♻️

Резервное копирование

Ежедневные бэкапы конфигураций. Возможность восстановления за 15 минут.
