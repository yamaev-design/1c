❓ Частые вопросы

Можно ли работать из другого города / страны?

Да, мы полностью удалённая компания. Главное — стабильный интернет и возможность работать в часовом поясе МСК ±3 часа для синхронизации с командой.

Какой график работы?

Гибкий график в рамках 6-часового окна (10:00–16:00 МСК) для синхронизации. Важно быть на связи в рабочее время, остальное — на ваше усмотрение.

Есть ли испытательный срок?

Да, 3 месяца. В это время вы получаете полную поддержку, регулярную обратную связь и возможность скорректировать задачи. Зарплата не меняется.

Какие технологии используете?

1С:Предприятие 8.3, Python для бэкенда, React для фронтенда, YandexGPT/GigaChat для ИИ, PostgreSQL, Docker. Полный стек — в описании вакансии.

Можно ли совмещать с другой работой / учёбой?

Для полных ставок — нет, мы ожидаем полной вовлечённости. Для стажировок и частичной занятости — да, обсудим индивидуальный график.

Не нашли подходящую вакансию?

Мы всегда открыты к талантливым людям. Отправьте резюме в общий пул — свяжемся, когда появится подходящая роль.
