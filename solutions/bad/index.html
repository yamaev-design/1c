❓ Частые вопросы продавцов БАД

Как ИИ помогает с описанием составов БАД?

Модуль автоматически извлекает ингредиенты из 1С и формирует понятные описания: «Омега-3 — поддержка сердца», «Витамин D — укрепление костей». Также выделяет противопоказания и формирует предупреждения для карточки, что снижает возвраты из-за непонимания состава.

Поддерживает ли решение сертификацию ЕАС?

Да, модуль автоматически формирует описания для декларирования БАД по ТР ТС 021/2011, генерирует тексты для сертификатов соответствия. Интеграция с реестрами Росаккредитации. Настройка занимает 10 минут через мастер подключения.

Как работает контроль сроков годности?

Модуль отслеживает сроки годности из 1С и автоматически скрывает товары с истекающим сроком из выгрузки на маркетплейсы. Также формирует уведомления для менеджеров за 90/60/30 дней до истечения срока, позволяя вовремя сделать скидку или вывести товар из продажи.

Можно ли генерировать описания для разных форм выпуска?

Да, ИИ создаёт уникальные описания для каждой вариации товара (капсулы, таблетки, порошок, жидкость), избегая дублирования. Например: «Омега-3 в капсулах» и «Омега-3 в жидкой форме» будут иметь разные тексты с релевантными ключами для каждой формы.

Поддерживает ли решение маркировку «Честный ЗНАК»?

Да, модуль интегрируется с системой маркировки для БАД: автоматическая выгрузка кодов, контроль остатков, формирование документов. Поддерживаются все категории БАД, подлежащие маркировке. Настройка занимает 10 минут через мастер подключения.
